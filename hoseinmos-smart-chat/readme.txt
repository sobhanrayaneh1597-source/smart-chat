=== Smart Chat === Contributors: hoseinmos Tags: chat, widget, ai, support, customer-service Requires at least: 6.5 Tested up to: 6.5 Requires PHP: 8.1 Stable tag: 1.0.0 License: GPLv2 or later License URI: https://www.gnu.org/licenses/gpl-2.0.html A smart chat widget that provides intelligent responses using internal site content and external sources. == Description == Smart Chat is a powerful WordPress plugin that adds an intelligent chat widget to your website. The widget can provide responses based on your website content, external sources, or a combination of both. = Features = * Floating chat widget with customizable position * Internal content search (posts, pages, custom post types) * External source integration (Bing, SerpAPI, OpenAI) * Hybrid mode combining internal and external results * Customizable appearance and colors * RTL support for Persian and Arabic languages * Rate limiting and security features * Admin panel for easy configuration * Privacy controls and logging options = Installation = 1. Upload the plugin files to the `/wp-content/plugins/smart-chat` directory, or install the plugin through the WordPress admin screen. 2. Activate the plugin through the "Plugins" screen in WordPress. 3. Go to Settings > Smart Chat to configure the plugin. = Frequently Asked Questions = = Can I use this without external API keys? = Yes! The plugin includes a mock provider that works without any external API keys for testing and demonstration purposes. = Is my data secure? = Yes, the plugin follows WordPress security best practices and includes rate limiting to prevent abuse. = Can I customize the appearance? = Yes, you can customize colors, position, and text through the admin panel. = Does it support RTL languages? = Yes, the plugin includes full RTL support for Persian, Arabic, and other RTL languages. = Changelog = = 1.0.0 = * Initial release * Basic chat widget functionality * Internal content search * Mock provider for testing * Admin configuration panel * RTL support = Upgrade Notice = = 1.0.0 = Initial release of Smart Chat plugin.
