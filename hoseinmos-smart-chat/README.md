# Smart Chat WordPress Plugin A smart chat widget that provides intelligent responses using internal site content and external sources. ## Features - Floating chat widget with customizable position - Internal content search (posts, pages, custom post types) - External source integration (Bing, SerpAPI, OpenAI) - Hybrid mode combining internal and external results - Customizable appearance and colors - RTL support for Persian and Arabic languages - Rate limiting and security features - Admin panel for easy configuration - Privacy controls and logging options ## Installation 1. Upload the plugin files to `/wp-content/plugins/smart-chat/` 2. Activate the plugin through the WordPress admin screen 3. Go to Settings > Smart Chat to configure the plugin ## Development ### Prerequisites - Node.js 16+ - npm or yarn ### Setup 1. Clone the repository 2. Run `npm install` to install dependencies 3. Run `npm run dev` for development 4. Run `npm run build` to build for production 5. Run `npm run zip` to create the installable ZIP file ### Alternative Build (No Node.js required) Run `./build.sh` to copy source files directly to assets directory. ## Configuration The plugin can be configured through the WordPress admin panel at Settings > Smart Chat. ### General Settings - Enable/disable the chat widget - Position (bottom-right, bottom-left) - Welcome text and placeholder - Theme colors ### Data Sources - Mode: internal, external, or hybrid - Internal content limit - External content limit - Mix weight for hybrid mode ### External Providers - Mock (for testing) - Bing Search API - SerpAPI - OpenAI API ### Privacy & Logging - Enable/disable chat logs - Log retention period - Rate limiting settings ## API Endpoints The plugin provides a REST API endpoint: `POST /wp-json/hmsc/v1/ask` Request body: `{ "message": "Your question here" }` Response: `{ "answer": "Response text", "sources": [...], "response_time": 0.123 }` ## Security Features - Rate limiting per IP address - Nonce verification for requests - Input sanitization and validation - SQL injection protection - XSS protection ## RTL Support The plugin includes full RTL support for Persian, Arabic, and other RTL languages. ## Testing Run `npm test` to execute the test suite. ## License GPL-2.0-or-later ## Author hoseinmos
